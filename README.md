# 🎓 R-Py 문제은행 풀이 프로그램

** R과 python 관련 문제를 제공하는 CLI(Command Line Interface) 기반의 문제 풀이 프로그램입니다.  
방대한 문제은행 데이터를 효율적으로 학습할 수 있도록 **랜덤 모의고사** 기능과 **순차 학습** 기능을 모두 제공합니다.

---

## ✨ 주요 기능 (Features)

### 1. 두 가지 풀이 모드 지원
* **🎲 랜덤 모드 (Random Mode):** 전체 문제은행에서 원하는 개수만큼 무작위로 문제를 뽑아 실전 모의고사처럼 풀 수 있습니다.
* **📚 순차 모드 (Sequential Mode):** 특정 번호부터 시작하여 원하는 개수만큼 차례대로 문제를 풉니다. (예: "100번부터 50문제 풀기")

### 2. 스마트한 학습 보조
* **즉각적인 정답 확인:** 문제를 풀자마자 정답/오답 여부를 알려줍니다.
* **진행 상황 파악:** 현재 문제 번호와 전체 진행률을 실시간으로 보여줍니다.
* **중간 점수 확인:** 문제 풀이 도중 `s` 또는 `score` 명령어로 현재 점수와 정답률을 확인할 수 있습니다.

### 3. 오답 노트 (Review System)
* 퀴즈가 끝나면 틀린 문제들만 모아서 **다시 풀기(Retry)** 기능을 제공합니다.
* 틀린 문제는 원본 문제 번호를 기억하고 있어, 나중에 교재나 자료에서 찾아보기 쉽습니다.

### 4. 안전한 데이터 처리
* 학습 도중 `q` 또는 `exit`를 입력하면 언제든지 안전하게 종료하며, 그때까지의 기록을 보여줍니다.
* 잘못된 입력(문자 입력 등)에 대한 예외 처리가 되어 있어 프로그램이 꺼지지 않습니다.

## 🛠 설치 및 실행 방법 (Getting Started)

이 프로그램은 Python 기본 라이브러리만 사용하므로 별도의 설치 과정이 필요 없습니다.

### 1. 필수 조건
* Python 3.x 이상이 설치되어 있어야 합니다.

터미널(또는 CMD)에서 프로젝트 폴더로 이동한 후 아래 명령어를 입력하세요.

python main.py
# 또는 파일명에 따라
